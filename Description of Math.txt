mn - числовой литерал

mo - для оператора (Например +, -)

mi - для переменных, функций, констант

msup - для прикрепления верхнего индекса к выражению //после выражения
	Праметры: base superscript
	Атрибуты: superscriptshift, class, id, style, href, mathbackground, mathcolor

msub - для прикрепления нижнего индкса к выражению
	Праметры: base subscript

msubsup - для прикрепления как нижнего, так и верхнего индекса к варыжению
	Параметры: base subscript superscript

mfenced - для настраивыемых открытых и закрытых скобок и разделителях в выражениях
	Атрибуты: open, close, separators

mfrac - для рисования дробей
	Параметры: числитель знаменатель
	Атрибуты: linethickness, numalign, denomalign, bevelled 

msgroup - умножение

mover - надстрочный индекс  //над всем выражением

munder - подстрочный индекс //над всем выражением

mpadded - для добавления отступов  

msqrt - квадратный корень  

mroot - радикалы с индексами
	Параметры: base index
